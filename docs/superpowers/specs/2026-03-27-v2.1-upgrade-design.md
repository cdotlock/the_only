## Summary

Based on a comprehensive review by a Claude engineer, this upgrade addresses all P0/P1 issues, adds Discord bot deep integration with feedback loop closure, improves onboarding UX for non-technical users, and polishes structural quality.

## Phase 1: P0 Bug Fixes

### 1.1 SR Card CSS 3D Flip
- Replace `display:none/block` toggle with real CSS 3D transform
- Add `.sr-card-inner` wrapper with `transform-style: preserve-3d`
- File: `references/webpage_design_guide.md`

### 1.2 knowledge_graph.py Path Hardcoding
- Add `--memory-dir` argument (default `~/memory/`)
- Derive `GRAPH_FILE` from argument instead of hardcoded path
- Update all CLI examples in SKILL.md
- File: `scripts/knowledge_graph.py`, `SKILL.md`

### 1.3 mesh_sync.py Pure Stdlib secp256k1
- Replace `import coincurve` with pure Python secp256k1 implementation
- Implement BIP-340 Schnorr sign/verify
- Validate against official Nostr test vectors
- File: `scripts/mesh_sync.py`

### 1.4 Google Fonts Fallback
- Add system font fallback stack to CSS variables
- Include CJK font families (Noto Serif SC, Source Han Sans)
- File: `references/webpage_design_guide.md`
